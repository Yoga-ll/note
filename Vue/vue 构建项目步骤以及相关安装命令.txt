安装淘宝镜像：npm install -g cnpm --registry=https://registry.npm.taobao.org

创建项目：vue init webpack projectname（在cmd中进入指定的目录输入命令3.0一以下） vue create projectname（3.0以上）
			
安装依赖包：npm install

卸载vue：npm uninstall @vue/cli -g

安装vue：npm install -g @vue/cli

启动项目：vue run serve






从零安装步骤：
1、下载Node.js--从官网去下载（检测node -v 和 npm -v ）
2、npm安装
	npm安装淘宝镜像：npm install -g cnpm –-registry=https://registry.npm.taobao.org
	npm安装vue-cli：cnpm install -g @vue/cli
3、创建项目--vue ui可视化窗口创建
